复平面的拓扑 Topology of the Complex Plane

邻域 Neighbourhoods

复数$z_0$的$\varepsilon$ 邻域, 也叫 开球 或者 开盘, 包括所有在以$z_0$为圆心以$\varepsilon>0$为半径的圆内部但不在圆上的所有点，用 \begin{eqnarray} B_{\varepsilon}(z_0)=\{z:|z-z_0| \lt \varepsilon\} \end{eqnarray} 表示。 $z_0$的闭$\varepsilon$邻域 用 \begin{eqnarray} \overline{B}_{\varepsilon}(z_0)=\{z:|z-z_0|\leq \varepsilon\} \end{eqnarray} 表示。 最后，$z_0$的删除 $\varepsilon$ 邻域 ，也叫做穿孔球或盘, 用 \begin{eqnarray} B_{\varepsilon}(z_0)\setminus \{z_0\}=\{z:0 \lt |z-z_0| \lt \varepsilon\} \end{eqnarray}表示。 图1显示了下列例子的几何表示：

  1. $B_{1}(0)=\{z:|z|\lt 1\}$
  2. $\overline{B}_{\frac{7}{8}}\left(-1-\sqrt{2}i\right)=\left\{z:|z-\left(-1-\sqrt{2}i\right)|\leq \frac{7}{8}\right\}$
  3. $\overline{B}_{\frac{1}{2}}\left(2+\sqrt{3}i\right)\setminus \left\{2+\sqrt{3}i\right\}=\left\{z:0 \lt |z-\left(2+\sqrt{3}i\right)|\leq\frac{1}{2}\right\}$

点的分类 Classification of points

设$z_0$为一个点，$S\subset \mathbb C$是一个集合，只要存在只包含$S$内点的$z_0$的邻域，我们就称$z_0$是$S$的内点。 如果存在不包含$S$中点的邻域，则称之为$S$的 外点。 如果这两点都不满足，则称$z_0$ 为$S$ 边界点 。 这样，边界点的任何邻域都至少包括一个$S$内的点和一个不在$S$内的点。所有边界点全体被称作$S$的 边界。

本文使用以下记号：

  1. $\text{Int } S=\left\{z: z \text{是} S \text{的内点}\right\}$
  2. $\text{Ext } S=\left\{z: z \text{ 是} S \text{的外点}\right\}$
  3. $\partial S=\left\{z: z \text{ 是} S \text{的边界点}\right\}$

考虑前面几个邻域的例子： $$ S_1=B_1(0),\; S_2=\overline{B}_{\frac{7}{8}}\left(-1-\sqrt{2}i\right) \text{ and } S_3=\overline{B}_{\frac{1}{2}}\left(2+\sqrt{3}i\right)\setminus \left\{2+\sqrt{3}i\right\}. $$ 这样，对$S_1$，有：

  • $\text{Int } S_1=B_1(0)$
  • $\text{Ext } S_1=\{z: |z|>1 \}$
  • $\partial S_1=\{z: |z|=1 \}$

对$S_2$,有：

  • $\text{Int } S_2=B_{\frac{7}{8}}\left(-1-\sqrt{2}\right)$
  • $\text{Ext } S_2=\left\{z:|z-\left(-1-\sqrt{2}i\right)|>\frac{7}{8}\right\}$
  • $\partial S_2=\left\{z:| z-\left(-1-\sqrt{2}i\right)| =\frac{7}{8}\right\}$

最后，对$S_3$有：

  • $\text{Int } S_3=B_{\frac{1}{2}}\left(2+\sqrt{3}\right)$
  • $\text{Ext } S_3=\left\{z:|z-\left(2+\sqrt{3}i\right)|>\frac{1}{2}\right\}$
  • $\partial S_3=\left\{z:|z-\left(2+\sqrt{3}i\right)|=\frac{1}{2}\right\}\cup\{0\}$

拓扑空间 Topological space

如果对每个$z\in S$，存在 $\varepsilon>0$使得 $$B_{\varepsilon}(z)\subset S.$$，则集合$S$是开的 。 也就是说$\text{Int } S= S$。 如果一个集合$S$包括其所有边界点，则该集合是闭的 ，也就是说 $$\partial S\subseteq S.$$ 如果一个集合$\mathbb C$ 没有边界点，则该集合即是开的，又是闭的。

集合 $\mathbb C$和集合组 $\tau =\{S\subseteq \mathbb C: S \text{是开的} \}$ 是一个 拓扑空间，用对$\left(\mathbb C, \tau\right)$表示。 拓扑空间$\left(\mathbb C, \tau\right)$要满足下列性质：；

  1. $\emptyset$ 和$\mathbb C$ 是开的。/li>
  2. 只要两个集合是开的，它们的并也是开的。
  3. 只要集合$S_1$ 和 $S_2$ 是开的，则 $S_1\bigcap S_2$也是开的。

备注： 拓扑空间的技术定义有点不太直观，特别是如果你没学过拓扑学的时候。 从本质上讲，它表达的是进行连续变换（函数或者映射）后集合$\mathbb{C}$的几何性质不变。

集合$S$的闭包是集合$S$中的所有点再加上$S$的边界。换句话说 $$\overline{S}=S\cup \partial S.$$

如果开集$S$内任意两点$z_1$ 和 $z_2$ 可以用全部位于集合内首尾相连的数量有限的线段组成的折线连起来，则该集合被称为 连通的。

例如，开集$|z| \lt 1$ 是连通的， 而环 $1 \lt |z| \lt 2$ 是开集，也是连通的，如图4 和 5所示。

连通的非空开集被称作 域。在这个前提下，任何邻域都是域。域和其部分、所有边界或者域本身被称作区间。换句话说，内部是域的集合被称作 区间。 如果存在 $R>0$ 使得 $$S\subset B_R(0)=\{z\in \mathbb C: |z|\lt R\}$$，则集合$S$是有限的。

  • $-1\lt \text{Re(z)} \lt 1$
